# Project A.TT.I.L.A. v0.1 Implementation Plan

> **For Gemini:** REQUIRED SUB-SKILL: Use superpowers:executing-plans to implement this plan task-by-task.

**Goal:** Build the v0.1 local-first PoC of Project A.TT.I.L.A., allowing a local Docker-based Gemini CLI agent (`spapparo`) to perform GCP discovery on `sre-next` with read-only permissions and save state in a sticky local memory folder.

**Architecture:** A Python CLI tool (`attila`) initializes the environment (creating `./memory/<PROJECT_ID>/` directories and `.env` config). A Docker container (`spapparo`) runs the `@google/gemini-cli` harness, mounts the host's `./memory` and a restricted Service Account key, authenticates `gcloud`, installs the SRE extension, and executes the discovery prompt.

**Tech Stack:** Python 3 (CLI), Node.js / Docker (Harness), `@google/gemini-cli` (Agent).

### Task 3: Create Dockerfile and Entrypoint for `spapparo`

**Files:**
- Create: `/usr/local/google/home/ricc/git/attila/Dockerfile`
- Create: `/usr/local/google/home/ricc/git/attila/entrypoint.sh`

**Step 1: Write `entrypoint.sh`**
A script that:
1. Checks for the mounted `/etc/gcp/sa-key.json`.
2. Authenticates `gcloud`:
   ```bash
   gcloud auth activate-service-account safe-sre-investigator@$PROJECT_ID.iam.gserviceaccount.com --key-file=/etc/gcp/sa-key.json
   gcloud config set project $PROJECT_ID
   ```
3. Installs the SRE extension:
   ```bash
   gemini extensions install https://github.com/gemini-cli-extensions/sre
   ```
4. Executes the `gemini` command with the provided prompt.

**Step 2: Write `Dockerfile`**
*   Base image: `node:24-bullseye` (or similar stable Debian-based image).
*   Installs Python, `gcloud` CLI (following official Google Cloud SDK installation steps), and `@google/gemini-cli`.
*   Copies `entrypoint.sh` and sets it as the entrypoint.

### Task 4: Create the `justfile` for easy orchestration

**Files:**
- Create: `/usr/local/google/home/ricc/git/attila/justfile`

**Step 1: Write `justfile`**
Include recipes:
*   `init PROJECT_ID`: Runs `python3 attila.py init --project-id {{PROJECT_ID}}`
*   `docker-build`: Builds the Docker image `attila:v0.1.0`.
*   `run-discovery PROJECT_ID`: Runs the docker container, mounting:
    *   `-v $(pwd)/memory/{{PROJECT_ID}}:/memory`
    *   `-v ~/git/gemini-cli-tools/tools/gcp/service-account-key.json:/etc/gcp/sa-key.json:ro`
    And passing env vars: `PROJECT_ID`, `GEMINI_API_KEY`.
